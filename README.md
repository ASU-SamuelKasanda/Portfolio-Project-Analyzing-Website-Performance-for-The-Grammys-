# Grammy.com Website Performance Analysis

## Overview
A Python-based data analytics portfolio project completed through the Global Career Accelerator in partnership with The Recording Academy (Grammys). This project analyzed web analytics data for Grammy.com and RecordingAcademy.com to evaluate the impact of a website split across two domains on user engagement and audience retention.

## Tools & Technologies
- Python (Pandas, Plotly Express)
- Jupyter Notebook
- Web Analytics Data Analysis
- Data Visualization

## Dataset
Web analytics dataset for Grammy.com and RecordingAcademy.com covering traffic volume, user engagement metrics, content performance, session behavior, and audience retention patterns across multiple content categories and time periods.

## What I Did
- Cleaned and processed multi-source web analytics data across 2 domains
- Analyzed user engagement metrics to measure the impact of the platform migration on audience behavior
- Identified content drop-off points and retention patterns across content categories
- Built data visualizations with Plotly Express to communicate findings clearly
- Synthesized insights into strategic recommendations for content and UX optimization

## Key Results
- Identified key user engagement gaps caused by the domain split across Grammy.com and RecordingAcademy.com
- Surfaced content categories with highest and lowest retention rates
- Delivered actionable content strategy and UX recommendations to improve digital audience growth and retention
